Transparenz ist uns sehr wichtig. Für typische Leistungen bieten wir Pauschalangebote an. Bei komplexeren Projekten wie einer Badsanierung erhalten Sie von uns eine detaillierte Kostenschätzung, die alle Schritte von der Planung bis zur Montage umfasst. Es gibt keine versteckten Kosten.
